# ریپوزیتوری کلاس پایتون دانشگاه

این ریپوزیتوری شامل کدهای تمرینی من از کلاس پایتون دانشگاه است.

## ساختار جلسات

### جلسه ۱ – متغیرها، انواع داده و شرط if/else
مواردی که یاد گرفتیم:
- تعریف متغیرها
- مقایسه متغیرها با شرط `if` و `else`
- انواع داده‌ها: عدد صحیح (`int`)، عدد اعشاری (`float`)، رشته (`str`)
- تغییر مقدار متغیر و بررسی نوع آن با `type()`

***
### جلسه ۲ – لیست‌ها و متودهای آن
مواردی که یاد گرفتیم:
- تعریف و ساخت لیست‌ها
- دسترسی و تغییر اعضای لیست
- متودهای لیست: `append`، `extend`، `insert`، `remove`، `pop`، `clear`، `sort`
- برش لیست (slicing) و استفاده از اندیس منفی
- حلقه `for` برای دسترسی به همهٔ عناصر لیست
